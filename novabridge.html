Blender and DCC Workflow

Pipeline: AI Agent → Blender scripts/extensions → OBJ/FBX export → /nova/asset/import → UE5 placement and automation.

NovaBridge supports configurable Blender integration via environment variables (path, export/output dirs, host/port, import scale, optional API key) for macOS, Windows, and Linux workflows.

This is a key differentiator for teams that want autonomous content generation plus direct in-editor orchestration.

Architecture + Security

Flow: AI → HTTP → Unreal Editor

Default binding: 127.0.0.1:30010

Port override: -NovaBridgePort=<port>

Optional auth: API key via NOVABRIDGE_API_KEY or -NovaBridgeApiKey with X-API-Key / Bearer header.

CORS: included on responses for tooling/UI interoperability.
