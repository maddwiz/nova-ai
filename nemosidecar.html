What It Adds To OpenClaw

Plan Creation

Create pending multi-step plans for desktop, browser, and operator workflows before taking action.

Guarded Approvals

Keep execution preview-only by default and approve only the plans you actually want to run.

Browser Readiness

Check whether the browser runtime is available before leaning on automation-heavy plans.

One-Click Sidecar Install

Install just the sidecar bundle without forcing the full NovaSpine memory stack.

Built On Full NovaAdapt

NemoSidecar is the commercial OpenClaw product surface for NovaAdapt. The current shipped layer is curated on purpose, but it can be expanded into higher tiers with broader execution, deeper tool coverage, and more operator controls.

That makes it a strong candidate for tiered pricing: simple by default, more powerful as you move up.

Install Shape

macOS: Install NovaAdapt Sidecar.command

Linux: Install NovaAdapt Sidecar.sh

Windows: Install NovaAdapt Sidecar.cmd

Works standalone. Pairs especially well with NovaSpine Pro if you also want long-term memory and fact governance.
